Beginner CSS

1. Introduction to CSS

What is CSS?: Explain that CSS stands for "Cascading Style Sheets" and is used to control the layout and appearance of web pages.

How CSS Works: Explain how CSS is used alongside HTML to style the content on a webpage.

Ways to Include CSS:

  • Inline CSS: Styling directly within an HTML element.
  • Internal CSS: CSS within a <style> tag in the HTML document.
  • External CSS: Linking to an external .css file.

2. Basic Syntax

CSS Rule Structure: Show how a CSS rule consists of a selector, property, and value.

        selector {
            property: value;
        }

Example:

        h1 {
            color: blue;
        }

Selector: Targets the HTML element.

Property: Defines the styling (e.g., color).

Value: The value for the property (e.g., blue).

3. Selectors

Basic Selectors:

  • Element Selectors: Selects HTML tags (e.g., <h1>, <p>).
  • Class Selectors: Targets elements with a specific class using the . (dot) notation (e.g., .myClass).
  • ID Selectors: Targets elements by their id using # (hash) notation (e.g., #myId).

Combining Selectors:

  • Grouping Selectors: Applying the same styles to multiple elements (e.g., h1, h2, h3).
  • Descendant Selectors: Targeting an element inside another element (e.g., div p).

4. CSS Box Model

Introduce the Box Model, which is fundamental to layout and spacing.

  • Content: The actual content inside the box.
  • Padding: Space between the content and the border.
  • Border: Surrounds the padding (or content if no padding).
  • Margin: Space outside the border.

Example Diagram:

        |          Margin             |
        | +------------------------+  |
        | |        Border           |  |
        | | +--------------------+ |  |
        | | |     Padding         | |  |

        | | | |  Content        |  | |  |
        | | | +----------------+  | |  |
        | | +--------------------+ |  |
        | +------------------------+  |
        +----------------------------+

5. Colors and Backgrounds

Setting Colors: How to apply colors using different formats:

  • Named Colors (e.g., red, blue).
  • Hexadecimal (#ff0000 for red).
  • RGB (rgb(255, 0, 0) for red).
  • HSL (hsl(0, 100%, 50%) for red).

Backgrounds:

  • Background Color: background-color.
  • Background Images: background-image.
  • Background Positioning: background-position, background-size.

6. Typography

Font Properties:

  • font-family: Set the font type (e.g., Arial, Courier New).
  • font-size: Control the size of the text (e.g., 16px, 1.5em).
  • font-weight: Control the thickness of the text (bold, normal, lighter).
  • font-style: Set italic or normal text (italic, normal).

Text Styling:

  • color: Set text color.
  • text-align: Control text alignment (left, center, right).
  • line-height: Adjust space between lines.
  • text-decoration: Control underlining or strike-through (e.g., underline, none).

7. Margins, Padding, and Borders

  • Margin: How to use margin to create space around elements.
  • Padding: How to use padding to create space within elements.
  • Borders: Adding borders to elements (border: 1px solid black).
  • Shorthand Properties: Using shorthand for setting margins, padding, and borders.

8. Width, Height, and Display Properties

Setting Dimensions:

  • width: Control the width of an element.
  • height: Control the height of an element.

Display Property:

  • block: Forces the element to take up the full width.
  • inline: Keeps elements inline with others.
  • inline-block: A combination of inline and block.
  • none: Hides the element from the page.

9. Positioning Elements

  • Static: Default positioning.
  • Relative: Position relative to its normal position.
  • Absolute: Position relative to the nearest positioned ancestor.
  • Fixed: Positioned relative to the viewport (doesnâ€™t scroll with the page).
  • Z-index: Control the layering of elements on top of each other.

Intermediate CSS

1. CSS Specificity

CSS specificity determines which CSS rule is applied by the browser when multiple rules target the same element. Specificity is calculated based on the types of selectors used.

  • Inline Styles: Highest specificity.
  • ID Selectors: Higher specificity than class and element selectors.
  • Class Selectors: Moderate specificity.
  • Element Selectors: Lower specificity.

Example:

        #myId {
            color: red;
        }
        .myClass {
            color: blue;
        }
        p {
            color: green;
        }

If a paragraph element (<p>) has both a class and an ID applied to it, the ID selector will take precedence.

2. The Cascade and Inheritance

The cascade is how CSS applies rules according to importance (source order) and specificity. Some properties are inherited from parent elements, and others are not.

  • Cascade: Determines which rule wins when rules conflict.
  • Inheritance: Certain properties, such as color and font-family, are inherited by child elements, while others like padding and margin are not.

3. Pseudo-classes and Pseudo-elements

Pseudo-classes and pseudo-elements allow you to style elements based on their state or add styles to parts of an element.

  • Pseudo-classes: Target elements in a specific state (e.g., :hover, :focus, :nth-child).
  • Pseudo-elements: Style specific parts of elements (e.g., ::before, ::after, ::first-line).

Example of pseudo-class:

        a:hover {
            color: red;
        }

Example of pseudo-element:

        p::first-line {
            font-weight: bold;
        }

4. CSS Positioning

Understanding CSS positioning is essential for controlling the layout of elements.

  • Relative Positioning: Moves an element relative to its original position.
  • Absolute Positioning: Moves an element in relation to the nearest positioned ancestor (not static).
  • Fixed Positioning: Fixes an element relative to the viewport (it stays in place while scrolling).
  • Sticky Positioning: Toggles between relative and fixed positioning, depending on the scroll position.

Example:

        div {
            position: absolute;
            top: 50px;
            left: 100px;
        }

5. CSS Transitions

CSS transitions allow you to change property values smoothly over a specified duration.

Basic Syntax:

        element {
            transition: property duration timing-function delay;
        }

Example:

        button {
            background-color: blue;
            transition: background-color 0.5s ease;
        }
        button:hover {
            background-color: green;
        }

6. CSS Animations

CSS animations allow you to create more complex animations than transitions by defining keyframes.

Basic Syntax:

        @keyframes myAnimation {
            0% {
                opacity: 0;
                transform: translateY(-50px);
            }
            100% {
                opacity: 1;
                transform: translateY(0);
            }
        }
        
        element {
            animation: myAnimation 2s ease-in-out infinite;
        }

7. CSS Flexbox

The Flexbox layout is designed to help you distribute space and align items within a container.

  • Flex Container: The parent element with display: flex;.
  • Flex Items: The children of the flex container.

Flex Properties for the Container:

  • flex-direction: Defines the direction of the flex items (row, column).
  • justify-content: Aligns items horizontally (start, center, space-between).
  • align-items: Aligns items vertically (stretch, center, flex-start).

Example:

        .container {
            display: flex;
            justify-content: center;
            align-items: center;
            height: 100vh;
        }

8. CSS Grid

CSS Grid is a layout system designed for two-dimensional layouts, allowing you to position elements into rows and columns.

Basic Syntax:

        .container {
            display: grid;
            grid-template-columns: 1fr 2fr 1fr;
            grid-template-rows: auto;
            grid-gap: 10px;
        }

Grid Properties:

  • grid-template-columns: Defines the number and size of columns.
  • grid-template-rows: Defines the number and size of rows.
  • grid-gap: Adds space between the grid items.

9. Media Queries

Media queries allow you to apply different styles depending on the device's screen size or capabilities.

Basic Syntax:

        @media (max-width: 600px) {
            body {
                background-color: lightblue;
            }
        }

10. Responsive Design

Responsive design ensures your website works well on different devices and screen sizes by using flexible layouts, media queries, and other techniques.

  • Fluid Layouts: Use relative units like percentages for widths.
  • Media Queries: Apply different styles based on device size or other features.
  • Flexible Images: Ensure images scale properly within their containers.

Advanced CSS

1. CSS Variables (Custom Properties)

CSS Variables (also known as custom properties) allow you to store values in one place and reuse them throughout your stylesheet, making it easier to maintain and update styles.

Basic Syntax:

            :root {
                --main-bg-color: #ff6347;
                --main-text-color: #333;
            }
            
            body {
                background-color: var(--main-bg-color);
                color: var(--main-text-color);
            }

In this example, the colors are stored in variables, and you can use them throughout the CSS using var().

2. Advanced CSS Selectors

CSS offers several advanced selectors that allow you to target elements with precision.

  • Attribute Selectors: Target elements based on the presence or value of attributes.
  • Child Selectors: Select direct children of an element (e.g., div > p).
  • Adjacent Sibling Selectors: Select the element that is immediately preceded by a specified element (e.g., h1 + p).
  • General Sibling Selectors: Select all elements that are preceded by a specified element (e.g., h1 ~ p).

Example:

            input[type="text"] {
                background-color: lightblue;
            }
            
            div > p {
                font-size: 1.2em;
            }
            
            h1 + p {
                color: red;
            }
            
            h1 ~ p {
                font-style: italic;
            }

3. CSS Grid Layout Advanced Techniques

CSS Grid Layout is one of the most powerful layout systems available in CSS, and there are advanced techniques that make it even more flexible.

Named Areas in Grid:

            .container {
                display: grid;
                grid-template-areas:
                    "header header header"
                    "nav content ads"
                    "footer footer footer";
                grid-gap: 10px;
            }
            
            .header {
                grid-area: header;
            }
            .nav {
                grid-area: nav;
            }
            .content {
                grid-area: content;
            }
            .ads {
                grid-area: ads;
            }
            .footer {
                grid-area: footer;
            }

4. CSS Transforms

CSS Transforms allow you to rotate, scale, skew, or translate elements. They provide a way to change the shape and position of an element.

Basic Syntax:

            element {
                transform: scale(1.5); /* Makes the element 50% larger */
                transform: rotate(45deg); /* Rotates the element 45 degrees */
                transform: translateX(20px); /* Moves the element 20px to the right */
            }

5. CSS 3D Transforms

CSS 3D transforms allow you to position elements in a three-dimensional space.

Basic Syntax:

            element {
                transform: perspective(400px) rotateX(45deg) rotateY(30deg);
            }

6. CSS Transitions and Animations (Advanced)

We can go beyond simple transitions with advanced animations. Using keyframes, you can create complex sequences of animations.

Multiple Animations:

            @keyframes fadeIn {
                from {
                    opacity: 0;
                }
                to {
                    opacity: 1;
                }
            }
            
            @keyframes moveDown {
                from {
                    transform: translateY(-100px);
                }
                to {
                    transform: translateY(0);
                }
            }
            
            element {
                animation: fadeIn 1s ease-in, moveDown 1s ease-out;
            }

7. CSS Shapes

CSS Shapes allow you to create non-rectangular layouts, where content wraps around shapes such as circles, ellipses, and polygons.

Example:

            img {
                float: left;
                shape-outside: circle(50%);
                width: 200px;
                height: 200px;
            }
            
            p {
                shape-inside: circle(50%);
            }

8. Clip Path

clip-path allows you to create complex shapes by clipping an element to a specific shape.

Basic Syntax:

            element {
                clip-path: circle(50% at 50% 50%);
                clip-path: polygon(50% 0%, 100% 100%, 0% 100%);
            }

9. CSS Filters

CSS Filters apply effects such as blur, brightness, contrast, and more to an element.

Example:

            img {
                filter: blur(5px) brightness(0.8);
            }

10. Custom Fonts and @font-face

To use custom fonts in your project, you can use the @font-face rule to load external font files.

Basic Syntax:

            @font-face {
                font-family: 'MyCustomFont';
                src: url('fonts/mycustomfont.woff2') format('woff2'),
                     url('fonts/mycustomfont.woff') format('woff');
            }
            
            body {
                font-family: 'MyCustomFont', Arial, sans-serif;
            }
